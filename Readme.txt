// Implementing a Java app searching books by connecting between the program and a SQL server

1.
Source files are available on request.

2.
In order to run the program, on the Windows Command Prompt, type 'java -jar booksearch.jar'

3.
Clicking one result row on the panel, detailed information of the specific book is shown up.

*********************************************************************************
If JDBC driver for MySQL is NOT installed in your computer,
Please save the file, mysql-connector-java-5.1.39-bin.jar or its higher version (https://dev.mysql.com/downloads/connector/j/5.1.html)
in the folder below.
C:\ProgramFiles\java\jdk1.xxx>jre>lib>ext
